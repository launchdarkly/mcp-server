Thank you for installing the MCP server!

This chart deploys the LaunchDarkly Model Context Protocol (MCP) server in **{{ .Values.transport }}** mode.

To check the status of the deployment:

  kubectl get pods -l app.kubernetes.io/instance={{ .Release.Name }}

The service is exposed on port {{ .Values.service.port }} inside the cluster.  If you are running with the default **ClusterIP** service, you can port-forward locally:

  kubectl port-forward svc/{{ include "mcp-server.fullname" . }} 2718:{{ .Values.service.port }}

Then connect your client to `http://localhost:2718/sse`.

Environment variables can be configured via the `values.yaml` file, e.g.

```yaml
env:
  LAUNCHDARKLY_API_KEY:
    secretKeyRef:
      name: launchdarkly-secret
      key: apiKey
```

Find more configuration options by inspecting `values.yaml`.
