# ContextSearch

## Example Usage

```typescript
import { ContextSearch } from "@launchdarkly/mcp-server";

let value: ContextSearch = {
  filter: "*.name startsWith Jo,kind anyOf [\"user\",\"organization\"]",
  sort: "-ts",
  limit: 10,
  continuationToken: "QAGFKH1313KUGI2351",
};
```
