# Size

The size of the flag event impact. Sizes are defined as: none (0%), small (0-20%), medium (20-80%), large (>80%)

## Example Usage

```typescript
import { Size } from "@launchdarkly/mcp-server/models/components";

let value: Size = "medium";
```

## Values

```typescript
"none" | "small" | "medium" | "large"
```
