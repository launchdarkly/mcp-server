# FlagLinkPost

## Example Usage

```typescript
import { FlagLinkPost } from "@launchdarkly/mcp-server";

let value: FlagLinkPost = {
  key: "flag-link-key-123abc",
  deepLink: "https://example.com/archives/123123123",
  title: "Example link title",
  description: "Example link description",
};
```
