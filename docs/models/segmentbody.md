# SegmentBody

## Example Usage

```typescript
import { SegmentBody } from "@launchdarkly/mcp-server";

let value: SegmentBody = {
  name: "Example segment",
  key: "segment-key-123abc",
  description: "Bundle our sample customers together",
  tags: [
    "testing",
  ],
  unbounded: false,
  unboundedContextKind: "device",
};
```
